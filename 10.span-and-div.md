# span 과 div

아무런 의미없이 어떤 태그들을 하나로 묶어서 스타일링을 해야할 때 div나 span 태그를 일종의 컨테이터로 사용한다.

## div

특정 태그들을 감싸 분할하는 블록레벨 요소이다.

특별하게 부여된 의미가 없다.

```html
<div class="box"></div>
```

## span

특정 문구들을 감싸 분할하는 인라인레벨 요소이다.

특별하게 부여된 의미가 없다.

```html
<p>안녕하<span class="bold">살법</span> 사랑<span class="italic">하문이다</span></p>
```
